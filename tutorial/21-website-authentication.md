# 21. Website Authentication

## Introduction

Website authentication is one of the trust-service-related areas connected to the eIDAS framework.

## Why It Matters

Website authentication is relevant because trust on the web is not only about identity at user level. It also concerns whether a website can be trusted as the claimed service or entity.

## Certificates

Website authentication is linked to certificate-based trust and, in some contexts, to qualified certificates within the framework.

## Summary

Website authentication is part of the broader digital-trust environment and contributes to trusted online interactions.
