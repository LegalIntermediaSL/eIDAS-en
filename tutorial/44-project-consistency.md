# 44. Overall Project Consistency

## Introduction

As a tutorial grows, one major risk is losing coherence across chapters.

## Consistency Principles

- legal categories should remain distinct
- technical tools should not be presented as if they were equivalent
- evidence should be treated as a chain, not just a final document
- didactic simplification should not erase core distinctions

## Summary

Project consistency depends on preserving distinctions while keeping the material readable.
